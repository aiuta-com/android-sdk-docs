Aiuta SDK 0.3.6 Help

Getting started with Aiuta

In this tutorial, you will learn how to initialize the SDK and figure out how to create your own application using all the features of Aiuta

Code example is here

The Aiuta SDK for Android provides the ability to use public methods provided by Aiuta from dev portal.

Prerequisites

Before starting this tutorial:

  • Install Android Studio

Create a new project

First of all, we need to create a project in which we will use the Aiuta SDK

  1. On the Welcome screen, click New Project. Otherwise, from the main menu, select File | New | New Project.

  2. Choose Empty Activity project

  3. Wait for synchronization of IDEA

Add dependencies

Let's add dependencies required for a Aiuta SDK.

  1. Be sure, that you use mavenCentral for solving dependencies in your root build.gradle.kts file

    repositories { mavenCentral() }

  2. Solve what the last version of Aiuta sdk on Github releases page

    You also can check last version of artifacts on Central Sonatype

  3. Open the app/build.gradle.kts file and add the following artifacts to the dependencies block:

    dependencies { val fashionVersion: String = "0.3.6" implementation("com.aiuta:fashionsdk:$fashionVersion") }

    Pay your attention on using the Bill of Materials

  4. Synchronize Project with gradle file

Initialize Aiuta

  1. Create MainApplication to initialize Aiuta entry point as follow:

    class MainApplication : Application() { lateinit var aiuta: Aiuta ... private fun initFashion() { aiuta = Aiuta.Builder() .setApiKey(API_KEY) .setApplication(this) .build() } ... }

    You can find your API key in Aiuta dev portal

    Be sure, that you set API key and Application, otherwise it will throw exception

  2. That's it! You are ready to explore all Aiuta possibilities
