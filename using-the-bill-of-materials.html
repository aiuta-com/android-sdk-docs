Using the Bill of Materials

In addition to the usual dependency plug-in, you can use Bill of Materials for better versioning of all Aiuta SDK dependencies

dependencies { val fashionVersion: String = "1.1.5" implementation(platform("com.aiuta:fashionsdk-bom:$fashionVersion")) implementation("com.aiuta:fashionsdk") // ... }

You also can check last version of BOM on Central Sonatype
