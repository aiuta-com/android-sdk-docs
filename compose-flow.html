Aiuta Try On flow

In this tutorial, you will learn how to use Aiuta Digital Try On power with Jetpack Compose Framework

Prerequisites

Before starting this tutorial:

  • Initialize Aiuta Try On Flows

Use Aiuta Digital Try On in Compose

Now you can use the prepared UI to try on the SKU. To do this, call the following in the appropriate place:

AiutaTryOnFlow( modifier = ..., aiuta = { ... }, aiutaTryOn = { ... }, aiutaTryOnListeners = { ... }, aiutaTryOnConfiguration = { ... }, theme = { ... }, skuForGeneration = { ... }, )

Where:

  • aiuta - is a instance of Aiuta for initializing of internal flow (code and api reference)

  • aiutaTryOn - is a instance of AiutaTryOn for using all logic of Aiuta Digital Try On (code and api reference)

  • aiutaTryOnListeners - is a instance of AiutaTryOnListeners for observing user interaction inside flow (code and api reference)

  • aiutaTryOnConfiguration - is a optional instance of AiutaTryOnConfiguration for configure internal behaviour of composable flow (code and api reference)

  • theme - is a optional theme decoration for AiutaTryOnFlow (code and api reference)

  • skuForGeneration - is SKU item for fitting on picked photo (code and api reference)
