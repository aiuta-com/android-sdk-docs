Getting started with Aiuta Analytic

In this tutorial, you will learn how to initialize AiutaAnalytic and observe AnalyticEvent from SDK

Prerequisites

Before starting this tutorial:

  • Initialize Aiuta SDK

Add dependencies

Let's add dependencies required for a Aiuta Analytic

  1. Firstly, add base dependencies, as described in Getting started with Aiuta guide

  2. Add analytic dependency to have access to AiutaAnalytic

    dependencies { val fashionVersion: String = "1.1.6" implementation("com.aiuta:fashionsdk-analytic:$fashionVersion") }

Get Aiuta Analytic

  1. Be sure, that your initialize Aiuta and have instance of this class. Otherwise, move to initialization guide and after that move back

  2. After that, you can use standard extension for creating new instance of AiutaAnalytic

    import com.aiuta.fashionsdk.Aiuta import com.aiuta.fashionsdk.analytic.AiutaAnalytic import com.aiuta.fashionsdk.analytic.analytic //... val aiuta: Aiuta = ... //... val aiutaAnalytic: AiutaAnalytic = aiuta.analytic

  3. You're ready to observe AnalyticEvent from SDK!

    aiutaAnalytic.analyticFlow.collect { newEvent -> // Do something with new event from SDK }
