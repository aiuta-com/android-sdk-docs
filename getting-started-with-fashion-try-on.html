Getting started with Aiuta Try On

In this tutorial, you will learn how to initialize AiutaTryOn powered by Aiuta Digital Try On

Code example is here

Prerequisites

Before starting this tutorial:

  • Initialize Aiuta SDK

Add dependencies

Let's add dependencies required for a Aiuta Try On

  1. Firstly, add base dependencies, as described in Getting started with Aiuta guide

  2. Add try on core dependency to have access to AiutaTryOn

    dependencies { val fashionVersion: String = "1.1.5" implementation("com.aiuta:fashionsdk:$fashionVersion") implementation("com.aiuta:fashionsdk-tryon-core:$fashionVersion") }

Get Aiuta Try On

  1. Be sure, that your initialize Aiuta and have instance of this class. Otherwise, move to initialization guide and after that move back

  2. After that, you can use standard extension for creating new instance of AiutaTryOn

    import com.aiuta.fashionsdk.Aiuta import com.aiuta.fashionsdk.tryon.core.AiutaTryOn import com.aiuta.fashionsdk.tryon.core.tryon //... val aiuta: Aiuta = ... //... val aiutaTryOn: AiutaTryOn = aiuta.tryon

  3. You're ready to make your first digital try on!
